This folder holds archived project clutter that was previously at the repository root.

- `manual-tests/`: ad hoc diagnostic scripts for background handling, calibration, and fitting experiments.
- `legacy-build/`: older PyInstaller specs and helper scripts that are not part of the current launcher/build path.
- `shelved-xcalibur-overlay/`: paused attempt to run the tool from the fixed `C:\Xcalibur\Python312` installation with an extra `runtime/site-packages` overlay.
- `prototypes/`: exploratory background-handling prototype code and debug plots.
- `generated/`: previous `build/`, `dist/`, and `__pycache__/` outputs.

The active source entry points remain at the repo root:
- `eds_tool.py`
- `eds_session.py`
- `intensity_table_dialog.py`
- `eds_tool_folder.spec`
